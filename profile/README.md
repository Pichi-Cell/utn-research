# Laboratorio de Sistemas Embebidos — UTN FRA

Bienvenido/a a la organización GitHub del **Laboratorio de Sistemas Embebidos** de la Facultad Regional Avellaneda, Universidad Tecnológica Nacional.

## ¿Qué hacemos?

Desarrollo de sistemas embebidos, firmware, hardware y herramientas para proyectos de investigación y formación académica.
